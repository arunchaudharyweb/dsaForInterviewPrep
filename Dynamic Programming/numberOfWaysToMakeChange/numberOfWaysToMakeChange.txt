Given an array of positive integers representing coin denominations and a single non-negative integer n 
representing a target amount of money, write a function that returns the number of ways to make change for that 
target amount using the given coin denominations.

Note that an unlimited amount of coins are at your disposal.

Sample Input:
  n = 6
  denoms = [1, 5]

Sample Output:
  2 // 1*1 + 1*5 and 6*1
